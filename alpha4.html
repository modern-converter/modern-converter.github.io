About Modern Forge

Modern Forge is a privacy-first, on-device file converter. It runs entirely in your browser, using built-in web APIs to transform files without uploading them to any server.

We focus on safety, speed, and beautiful UX. The app adapts to your device to keep memory usage safe and efficient.

Philosophy

We believe powerful tools can be simple, private, and accessible. Modern browsers provide the building blocks for media processing, and with thoughtful design, they can feel like native apps.

While some conversions are “lite” placeholders due to web limitations, they keep your data local and provide broadly compatible results.

Security & Privacy
  • Files never leave your device. All processing is done in the browser.
  • No account, no tracking, no analytics beacons.
  • Device-safe memory heuristics keep your system responsive.
  • Permissions are not required; features use sandboxed browser capabilities.
Tips
  • Prefer modern formats (WebP/AVIF for images, WebM for video) for smaller sizes.
  • Increase “Parallel conversions” on desktops for faster batch processing.
  • Reduce maximum size if you experience slowdowns on low-memory devices.
Quick Start
  1. Drag & drop files into the upload area.
  2. Choose the output format (the right one is auto-detected from your files).
  3. Click Convert to open the progress view. Download when done.
FAQ

Why “lite” formats? Some encoders are not natively available in browsers. We provide lightweight, standards-friendly placeholders that are broadly compatible.

Are my files safe? Yes. Everything is processed locally and never uploaded.
